## 一键安装

### 小狼毫98五笔版

- 内置字体支持
- 支持显分显示
- 支持精准造词

#### 开关及提示

- ctrl+shift+F = 繁简转换

- ctrl+shift+H = 拆分显隐

- ctrl+shift+J = 三重注解

- ctrl+shift+U = 字集过滤

- ctrl+shift+K = 单字模式

- help 进入帮助页
